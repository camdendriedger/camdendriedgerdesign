The Project
The project I was assigned to complete for a course was essentially to create multiples of some sort of media, in the theme of a game, in any medium.
CSS Animations
I took this project as an opportunity to explore in depth possibilites of inline SVG in conjunction with CSS animations. All imagery is inline SVG, generated/coded by hand and are animated with css hover animations.
